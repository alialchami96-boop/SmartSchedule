# SmartSchedule - نظام إدارة الجداول الدراسية الذكي

## نظرة عامة

SmartSchedule هو تطبيق Android احترافي لإدارة وإنشاء الجداول الدراسية بشكل تلقائي باستخدام الخوارزميات الجينية المتقدمة. يحل التطبيق مشكلة تعقيد إنشاء الجداول يدوياً من خلال أتمتة العملية بالكامل مع مراعاة جميع القيود والمتطلبات.

## المميزات الرئيسية

### إنشاء تلقائي للجداول
- خوارزمية جينية متقدمة مع 500 جيل تطوري
- معدل نجاح عالي في إيجاد حلول مثالية
- تحسين تكيفي يزيد من معدل التحور عند الركود
- نظام Elitism للحفاظ على أفضل الحلول
- Tournament Selection لاختيار أفضل الأفراد

### إدارة شاملة للبيانات
- إدارة المواد الدراسية: إضافة، تعديل، حذف مع ربط المدرسين
- إدارة المدرسين: تحديد التخصصات والحد الأقصى للساعات الأسبوعية
- إدارة الفصول: تنظيم الصفوف والأقسام مع تحديد عدد الطلاب
- إدارة القاعات: تحديد السعة والنوع والمعدات المتاحة

### مرونة في التخصيص
- أوزان قابلة للتعديل: 9 معايير مختلفة للجدولة
- إعدادات مسبقة: 4 أنماط جاهزة (متوازن، مركز على المدرسين، القاعات، الطلاب)
- حفظ تلقائي للإعدادات باستخدام DataStore
- إمكانية ضبط كل معيار بشكل منفصل

### تصدير متعدد الصيغ
- PDF: مناسب للطباعة والعرض الاحترافي
- Excel CSV: قابل للتحرير في برامج الجداول
- CSV: بيانات خام للتحليل الإحصائي

### واجهة مستخدم حديثة
- Material Design 3 مع مكونات حديثة
- دعم كامل للغة العربية في جميع الشاشات
- تصميم متجاوب يعمل على جميع أحجام الشاشات
- جاهز لوضع الليل Dark Mode

## البنية المعمارية

### Clean Architecture ثلاث طبقات

الطبقة الأولى - Presentation Layer:
- ViewModels لإدارة حالة التطبيق
- UI Screens مبنية بـ Compose
- Navigation للتنقل بين الشاشات

الطبقة الثانية - Domain Layer:
- Models نماذج منطق الأعمال
- UseCases حالات الاستخدام
- Repository Interfaces واجهات المستودعات

الطبقة الثالثة - Data Layer:
- Room Database قاعدة البيانات
- Repository Implementation تطبيق المستودعات
- DataStore لحفظ التفضيلات

### أنماط التصميم المستخدمة
- MVVM Pattern لفصل المنطق عن العرض
- Repository Pattern للوصول للبيانات
- Dependency Injection باستخدام Hilt
- Single Source of Truth لضمان تناسق البيانات
- Unidirectional Data Flow لتدفق أحادي
- Observer Pattern باستخدام StateFlow

## الأمان والخصوصية

### ProGuard Rules
- تشفير الكود عند البناء للإصدار النهائي
- حماية أسماء الفئات والدوال
- تقليل حجم ملف APK
- إزالة الكود غير المستخدم

### البيانات المحلية
- لا يتطلب اتصال بالإنترنت
- جميع البيانات محفوظة محلياً
- Room Database مشفرة افتراضياً
- DataStore آمن للإعدادات

## الخلاصة

SmartSchedule هو حل متكامل واحترافي يجمع بين قوة الخوارزميات الجينية وسهولة الاستخدام ومرونة التخصيص مع الالتزام بمعايير البرمجة الحديثة. التطبيق جاهز للاستخدام الفعلي في المؤسسات التعليمية ويوفر وقتاً وجهداً كبيرين مقارنة بالطرق التقليدية اليدوية.
